TrollStore2安装异常解决方案

发布于 2023-12-02 TrollStore2 

设备信息：

  • iphone14ProMax
  • ios 16.2
  • 开发者模式已打开

软件信息：

  • Misaka 3.2.8 beta
  • PersistenceHelper_Embedded
  • Tips （Store 最新版）
    异常：
    [修改 Tips 提示软件后软件不能运行，出现闪退]
    在使用 Misaka 正常安装的过程中，两种安装方式都不能成功：
  1. 点击 install trollstore 以后，Tips 打开一直闪退，删除重新安装 tips 重复步骤依然闪退。
  2. fileManger-var-Containers-Bundle-Application-com.apple.tips-Tips.app-Tips - file overwrite ,选中 PersistenceHelper_Embedded 文件，这里直接打开 tips 还是会闪退。
    解决方案：
    采用上门的 file overwrite 方式替换 PersistenceHelper_Embedded 文件以后，提示 file overwrite done 时对设备进行强制重启，依次按键：音量+、音量-、长按开机键直至苹果图标出现松开，这是再打开 tips 就不再闪退了，这里直接点击 install trollstore 即可正常安装，测试功能正常。

使用第一种方式安装时，强制重启是否生效，未测试，推测应该也是可以的。
